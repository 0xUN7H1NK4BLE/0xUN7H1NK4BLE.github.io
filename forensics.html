03 / Challenges

The Nepal Landscape

Key systematic bottlenecks currently encountered on the ground.

Statutory Limitations

The Electronic Transactions Act (ETA 2063) remains our central benchmark, but lacks provisions for contemporary challenges like localized cloud multi-tenancy and encrypted crypto assets.

Centralized Resources

Heavy platform licensing fees (Axiom, Cellebrite) limit specialized extraction suites to central labs, creating diagnostic backlogs for remote regional cases.

04 / Methodology

The Investigation Mindset

Preserving integrity by enforcing strict chain of custody protocols.

Volatile Data First

Collect unstable memory states (like live CPU caches & RAM) before acquiring offline persistent drives.

Analytical Copy Rule

Always generate a write-blocked bitstream image. Never run diagnostic tools on the original media.

07 / Forensic Branch

Computer & Disk Forensics

The collection, recovery, and analysis of static data stored inside physical storage sectors.

Registry Auditing:

Analyzing OS configuration databases to find system history, execution paths, and connected USB devices.

File System Inquest:

Reconstructing logs from systems like NTFS (MFT records) or ext4 (journal nodes) to find modified files.

08 / Forensic Branch

Network Forensics

Monitoring and analysis of network traffic to discover active intrusions, data leaks, and attacks.

Packet Capture (PCAP):

Capturing and parsing exact raw frame layers in transit (Ethernet, IP, TCP).

NetFlow Audits:

Studying transaction flows to map bandwidth patterns without logging raw payloads.

Mobile & Device Forensics

Logical vs. Physical Dump:

10 / Tools Suite

Forensic Acquisition Tools

Applications used to safely capture, clone, and preserve data with mathematical integrity.

GUI IMAGER

FTK Imager

The industry standard for quick disk imaging, directory mounting, and hash calculation. Known for reliability on Windows environments.

LINUX COMPLIANT

Guymager

A fast, open-source acquisition tool for Linux systems. Supports multi-threaded processing and writes out EWF (E01) files.

TERMINAL ENGINE

dd & dcfldd

Low-level command-line disk dumping utilities. `dcfldd` is customized for forensics, generating progress bars and on-the-fly hashes.

11 / Tools Suite

Investigation & Analysis Suites

All-in-one workspaces to analyze files, parse registries, index keywords, and trace history timelines.

OPEN SOURCE

Autopsy

A highly extensible, user-friendly open source interface for Sleuth Kit backend utilities. Excellent for analyzing registry files, EXIF, and carving deleted files.

ENTERPRISE CORE

Magnet AXIOM

A powerful platform specialized in consolidating memory, mobile applications, cloud repositories, and computer sectors into synchronized timeline views.

LEGACY BENCHMARK

EnCase Forensics

One of the oldest corporate suites, renowned for raw-sector scripting, court-verified structures, and comprehensive compliance capabilities.

12 / Tools Suite

Specialized Forensic Utilities

Targeted tools configured for specific, high-precision forensic investigations.

VOLATILE MEMORY

Volatility

The leading framework for command-line volatile RAM analysis. Identifies active processes, open network ports, and cached passwords in memory dumps.

METADATA READ / WRITE

ExifTool

An exceptionally versatile command-line utility for reading, writing, and parsing metadata across thousands of media file extensions.

NETWORK DECODER

Wireshark

The premier network protocol analyzer. Allows deep packet inspection, traffic decoding, and reconstruction of exfiltrated data layers.
