Aircrack-ng

It is a complete suite of tools to assess WIFI network security. Used for monitoring, attacking, testing, and cracking wireless networks. The good thing about it is It is an open-source tool. It is able to attack WPE networks as well as WPA and WPA2.

WEP network compromise.

We first need to put wifi card in monitor mode.

First, we need to kill all the services that could cause a problem so for that we can use airmon-ng.

sudo airmon-ng check && sudo airmon-ng check kill

now let’s turn on the monitor mode on a network card.

sudo airmon-ng start <card interface (wlan0)>

This will start monitor mode on the card now let’s search for all the available networks.

sudo airodump-ng wlan0mon

This will show all the networks around us.

Now let’s collect packets of the network.

airodump-ng wlan0mon --bssid <bssid of your target> -c (CH of target) -w capture

this will show all the traffic in the network, as well as connected devices, and -w, will save the packet in the capture file.

Now, we see that aircrak has created five files and the most interesting file is capture-01 with the .cap extension.

Finally, after capturing for a few minutes we can use aircrack-ng to crack password.

aircrack-ng capture-01.cap

This will give data in hex format but we can use it to connect the network.

Now, connect to the WEP network with the hex format password without colons, and boom you are connected to the network.

Now, let’s try the WPA-WPA2 network.

Most of the process is the same as WEP so start monitor mode and search for the network.

Now, find your attacker network with WPA2 and PSK auth and run the same command.

sudo airodump-ng wlan0mon -bssid <bssid of your target> -c (CH of target) -w capture

Now, we will see the connected devices and we have two options: kick the user out of the network or wait for other users to connect to the network for a handshake.

So, let’s run a deauthentication attack.

sudo aireplay-ng wlan0mon --deauth 20 -a <mac of wifi> -h <mac of pc connect in network>

This will send a de-authentication packet to the router pretending that we are victims.

Then the victim will be disconnected and if they reconnect we get a handshake and now we can stop the airdump-ng and crack the password using a wordlist.

aircrack-ng -w <wordlist> <wpa2 capture file with .cap ext>

And this will brute-force the password.

boom we got the password.
